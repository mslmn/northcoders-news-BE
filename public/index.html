Northcoders News API

    Overview Endpoints

This API provides access to Northcoders News articles, topics, and users.

Endpoints

GET /api/topics

Retrieves a list of all topics.

Example Request

GET /api/topics

Example Response

              "articles": [
                {
                  "article_id": 3,

GET /api/articles

Retrieves a list of all articles.

GET /api/users

Retrieves a list of all users.

GET /api/articles/:article_id

Retrieves a single article by its ID.

GET /api/articles/:article_id/comments

Retrieves comments for a specific article by its ID, sorted by date descending.
